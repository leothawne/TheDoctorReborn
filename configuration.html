♟ The Doctor Reborn ♟

A Minecraft RPG plugin based on Doctor Who.

List of settings from config.yml:

  • enable-plugin: true  #If for some reason you want to disable this plugin, just set 'enable-plugin' to 'false' (default: true).
  • use-worlds: true  #Change this to 'false' if you don't want to use TDR Worlds (default: true).
  • teleport-worlds: true #Change this to 'false' if you don't want to allow '/reborn tp' (default: true). This will not make any difference if 'use-worlds' is set to 'false'.
